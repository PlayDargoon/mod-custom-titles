## Custom Titles Module

Модуль для добавления кастомных званий в AzerothCore.

### Описание

Этот модуль позволяет создавать и управлять кастомными званиями для игроков. Звания могут иметь различные требования (уровень, достижения, предметы) и стоимость.

### Возможности

- ✅ Создание неограниченного количества кастомных званий
- ✅ Разные названия для мужских и женских персонажей
- ✅ Система требований (уровень, достижения, предметы)
- ✅ Стоимость в золоте
- ✅ Команды для игроков и администраторов
- ✅ Автоматическая загрузка при старте сервера
- ✅ Горячая перезагрузка званий без перезапуска сервера

### Установка

1. Скопируйте папку `mod-custom-titles` в директорию `modules/`
2. Перекомпилируйте сервер
3. Примените SQL файлы:
   - `data/sql/db-world/base/custom_titles.sql` к базе `acore_world`
   - `data/sql/db-characters/base/character_custom_titles.sql` к базе `acore_characters`
4. Скопируйте `conf/mod_custom_titles.conf.dist` в `conf/mod_custom_titles.conf` и настройте по желанию

### Использование

#### Команды для игроков:

- `.ctitle list` - Показать список доступных кастомных званий
- `.ctitle set <id>` - Установить кастомное звание
- `.ctitle remove` - Снять текущее кастомное звание

#### Команды для администраторов:

- `.ctitle reload` - Перезагрузить кастомные звания из базы данных

### Добавление новых званий

Добавьте новые записи в таблицу `custom_titles`:

```sql
INSERT INTO `custom_titles` (`id`, `name_male`, `name_female`, `required_level`, `required_achievement`, `required_item`, `cost`) 
VALUES (11, 'Ваше Звание', 'Ваше Звание', 80, 0, 0, 500);
```

После добавления используйте команду `.ctitle reload` для применения изменений.

### Конфигурация

В файле `mod_custom_titles.conf`:

```ini
# Включить/выключить модуль
CustomTitles.Enable = 1

# Объявлять звание при входе
CustomTitles.AnnounceOnLogin = 1
```

### Структура таблицы custom_titles

| Поле | Тип | Описание |
|------|-----|----------|
| id | INT UNSIGNED | Уникальный ID звания |
| name_male | VARCHAR(255) | Название для мужских персонажей |
| name_female | VARCHAR(255) | Название для женских персонажей |
| required_level | INT UNSIGNED | Требуемый уровень (0 = нет требования) |
| required_achievement | INT UNSIGNED | ID требуемого достижения (0 = нет требования) |
| required_item | INT UNSIGNED | ID требуемого предмета (0 = нет требования) |
| cost | INT UNSIGNED | Стоимость в золоте (0 = бесплатно) |
